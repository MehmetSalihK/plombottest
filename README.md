# plombot

A Discord Music Bot written in Python 3.8


## Running

1. Rename `keys.sample.py` to `keys.py` and fill in with your keys

1. Run the bot with Python 3

```bash
python3 -m pip install -r scripts/requirements.txt
python3 plombot.py
```
